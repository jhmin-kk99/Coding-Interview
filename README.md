##  __반복 학습__   
> * 한 단원이 시작되면, 이전 단원들의 복습을 함께 시작합니다   
> * 질문에 대한 답변을 생각해보고 답을 확인합니다     
> * 부족한 부분은 추가하여 공부합니다     
> * 이해, 원리 위주의 질문으로 구성합니다
